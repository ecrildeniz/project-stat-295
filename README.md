# Hotel Booking System

## Description
This is a Java-based hotel booking system that allows hotel staff to manage room availability, bookings, customer data, payments, and reviews. The application includes a graphical user interface (GUI) and demonstrates the core principles of Object-Oriented Programming (OOP):

- **Encapsulation**
- **Inheritance**
- **Polymorphism**
- **Abstraction**

## How to Run

> ⚠️ **Note:** Ensure that you have Java installed and added to your system's PATH.

1. Extract the ZIP file if you haven't already.
2. Double-click the `run.bat` file located in the project directory.
3. The hotel booking GUI will launch. You can use the menu options or buttons to view and manage:
   - Rooms
   - Customers
   - Bookings
   - Payments
   - Reviews

## Features

- Add and view room information  
- Manage customer and employee data  
- Create and view bookings  
- Handle payments  
- Submit and view customer reviews
